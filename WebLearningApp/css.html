CSS Tutorial

Definition

CSS (Cascading Style Sheets) is used to style HTML elements on a webpage. It controls layout, colors, fonts, and overall look.

Syntax

CSS uses selectors to target elements and apply properties:

selector {
    property: value;
}

Examples

Changing text color:

p {
    color: blue;
}

Background color:

body {
    background-color: #f0f0f0;
}

Text alignment:

h1 {
    text-align: center;
}

Inline, Internal, and External CSS

  • Inline CSS: Use the style attribute in HTML elements.
  • Internal CSS: Use <style> tag in <head>.
  • External CSS: Use a separate .css file linked via <link>.
