JavaScript Tutorial

Definition

JavaScript is a scripting language used to make web pages interactive. It runs in the browser and can change HTML and CSS dynamically.

Syntax

JavaScript uses statements, functions, and events. Example:

function greet() {
    alert("Hello World!");
}

Examples

Printing to console:

console.log("Hello World");

Changing HTML content:

document.getElementById("demo").innerHTML = "Hello JS";

Event Example:

<button onclick="alert('Clicked!')">Click Me</button>

Variables and Data Types

  • var / let / const - declare variables
  • Data types: string, number, boolean, object, array
  • Functions - perform actions and return values
